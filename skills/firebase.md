## Firebase Init (Varad does this once)
```dart
// lib/main.dart
import 'package:firebase_core/firebase_core.dart';
import 'firebase_options.dart'; // generated by flutterfire configure

void main() async {
  WidgetsFlutterBinding.ensureInitialized();
  await Firebase.initializeApp(options: DefaultFirebaseOptions.currentPlatform);
  runApp(const GramYatraApp());
}
```

## Setup Commands (run once)
```bash
dart pub global activate flutterfire_cli
flutterfire configure   # links Flutter to Firebase project
```

## Read POI Data
```dart
// lib/services/poi_service.dart
import 'package:cloud_firestore/cloud_firestore.dart';
import '../models/poi_model.dart';

class PoiService {
  final _db = FirebaseFirestore.instance;

  Future<List<PoiModel>> getPOIs() async {
    try {
      final snap = await _db.collection('poi').get();
      return snap.docs.map((d) => PoiModel.fromMap(d.data())).toList();
    } catch (e) {
      debugPrint('PoiService: $e');
      return PoiModel.mockData(); // fallback
    }
  }
}
```

## Anonymous Auth (Sahil does this)
```dart
// lib/services/auth_service.dart
import 'package:firebase_auth/firebase_auth.dart';

class AuthService {
  Future<void> signInAnonymously() async {
    try {
      await FirebaseAuth.instance.signInAnonymously();
    } catch (e) {
      debugPrint('AuthService: $e');
    }
  }
}
```
Call `signInAnonymously()` in `main()` after Firebase init.

## Firestore Offline Persistence (enable this)
```dart
FirebaseFirestore.instance.settings = const Settings(
  persistenceEnabled: true,
  cacheSizeBytes: Settings.CACHE_SIZE_UNLIMITED,
);
```
Add this right after `Firebase.initializeApp()`.
